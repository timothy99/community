## 1. 환경설정

경로: `/csl/config/view`

사이트 전반의 기본 정보 및 시스템 설정을 관리합니다.

| 항목 | 설명 |
|---|---|
| 회사명 | 사이트/회사 명칭 (필수) |
| 설명 | 사이트 설명 |
| 로고 | 회사 로고 이미지 |
| 전화번호 | 대표 전화번호 (필수) |
| 팩스 | 팩스 번호 |
| 이메일 | 대표 이메일 (필수) |
| 업무시간 | 운영 시간 안내 (필수) |
| 주소 | 우편번호, 기본주소, 상세주소 (필수) |
| 사업자등록번호 | 사업자등록번호 (필수) |
| 담당자 이메일 | 문의 등 알림을 받을 담당자 이메일 (필수) |
| 관리자 이메일 인증 | 관리자 로그인 시 이메일 2단계 인증 사용 여부 |
| 메일 발송 기능 | SMTP를 통한 메일 발송 사용 여부 |
| SMTP 설정 | 메일 서버 호스트, 포트, 계정, 비밀번호, 발송자명 |
| 프로그램 버전 | 현재 프로그램 버전 관리용 |

> 관리자 이메일 인증(`admin_two_factor_yn = Y`)을 사용하려면 SMTP 설정이 반드시 입력되어 있어야 합니다.

## 2. 다국어

경로: `/csl/language/edit`

> **메뉴에 한국어 외에 다국어 메뉴를 설정하려면 반드시 다국어를 사용으로 설정해야 합니다.**

사이트에서 사용할 언어를 설정합니다.

| 항목 | 설명 |
|---|---|
| 다국어 사용 여부 | 다국어 기능 활성화 여부 (사용/미사용) |
| 사용 언어 | 활성화할 언어 목록 선택 |

- 다국어를 사용으로 설정하면 메뉴구성에서 언어별 메뉴명을 별도로 입력할 수 있습니다.
- 미사용 시 메뉴 등록/수정 화면에서 다국어 입력란이 표시되지 않습니다.

## 3. 메뉴구성

경로: `/csl/menu/list`

사용자 화면에 노출되는 내비게이션 메뉴를 구성합니다.

| 항목 | 설명 |
|---|---|
| 메뉴명 | 화면에 표시될 메뉴 이름 (필수) |
| 링크 | 클릭 시 이동할 URL (필수) |
| 링크 타깃 | `_self` (현재 창) / `_blank` (새 창) |
| 메뉴 위치 | 상단/하단 등 메뉴 노출 위치 |
| 상위 메뉴 | 하위 메뉴인 경우 연결할 상위 메뉴 |
| 정렬 순서 | 같은 레벨 내 메뉴 표시 순서 (필수) |
| 언어 | 다국어 사용 시 해당 메뉴의 언어 구분 |

- 다국어가 활성화된 경우 같은 메뉴를 언어별로 각각 등록하여 다국어 메뉴를 구성합니다.
- 하위 메뉴 삭제 시 해당 메뉴만 삭제되며, 상위 메뉴는 영향을 받지 않습니다.

## 4. 콘텐츠

경로: `/csl/contents/list`

특정 URL에 연결하여 사용하는 정적 페이지(소개, 이용약관 등) 콘텐츠를 관리합니다.

| 항목 | 설명 |
|---|---|
| 콘텐츠 ID | 코드에서 콘텐츠를 호출할 때 사용하는 식별자 |
| 제목 | 콘텐츠 제목 (필수) |
| 메타 타이틀 | 브라우저 탭 및 SEO용 타이틀 |
| 내용 | 에디터를 통해 작성하는 본문 (HTML 가능) |

- 콘텐츠 ID를 통해 뷰에서 특정 콘텐츠를 불러와 출력할 수 있습니다.
- 검색 조건은 제목(`title`) 기준이며, 페이지당 노출 수를 선택할 수 있습니다.

## 5. 게시판 설정

경로: `/csl/settings/board/list`

게시판 생성 및 각 게시판의 세부 옵션을 설정합니다.

| 항목 | 설명 |
|---|---|
| 게시판 ID | 게시판을 구분하는 고유 식별자 (필수) |
| 제목 | 게시판 이름 (필수) |
| 타입 | 게시판 형태 (board 등) |
| 페이지당 표시 수 | 목록에서 한 페이지에 보여줄 게시글 수 (필수) |
| 카테고리 사용 | 카테고리 사용 여부 및 카테고리 항목 입력 |
| 회원 글쓰기 | 일반 회원의 글쓰기 허용 여부 |
| 댓글 쓰기 | 댓글 기능 사용 여부 |
| 조회수 표시 | 조회수 노출 여부 |
| 좋아요 | 좋아요(하트) 기능 사용 여부 |
| PDF 출력 | PDF 다운로드 기능 사용 여부 |
| 유튜브 | 유튜브 영상 첨부 기능 사용 여부 |
| URL 링크 | 외부 링크 첨부 기능 사용 여부 |
| 메인 이미지 | 대표 이미지 첨부 기능 사용 여부 |
| 파일 첨부 | 파일 첨부 기능 사용 여부, 최대 파일 수, 개당/전체 용량 제한 |
| 등록일 표시 | 글 목록에서 등록일 노출 여부 |
| 글쓰기 포인트 | 글 작성 시 적립 포인트 |
| 댓글 포인트 | 댓글 작성 시 적립 포인트 |
| 커스텀 폼 | 게시글 입력 폼을 HTML로 직접 정의하는 기능 |
| 권한 설정 | 목록/상세보기/글쓰기 각각 접근 가능한 권한 그룹 지정 |
